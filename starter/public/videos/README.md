# Demo recordings

Drop your screen recordings here. Each `<DemoVideo>` placeholder in `slides.md`
expects a file with the matching name — once the file exists, the slide swaps
the placeholder frame for a real video player automatically (no code change).

| File | Slide / task |
| --- | --- |
| `env-hook-guard.mp4` | The `.env` PreToolUse guard denying secret reads — auto-generated via VHS (see `tools/env-hook/`) |
| `hello-skill.mp4` | Create the `hello` skill, then type "alex" in chat |
| `install-skill-creator.mp4` | `npx skills add … --skill skill-creator` |
| `vue-ai-assistant.mp4` | Generate the `vue-ai-assistant` skill + ask a Vue question |
| `ask-questions.mp4` | `#askQuestions` clarifying-questions tool |
| `subagent-fanout.mp4` | `#runSubagent` → 3 parallel web-search agents |
| `excalidraw-brand.mp4` | Excalidraw skill restyled with brand colors |

Tip: `.mp4` (H.264 + AAC) plays everywhere. Keep clips short and silent or
narrate live over them.
